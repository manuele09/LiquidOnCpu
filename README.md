
# LiquidOnCpu

A C implementation of a Spiking Neural Networks (SNN).


## Run Locally

Clone the project

```bash
  git clone https://github.com/manuele09/LiquidOnCpu
```

Go to the project directory

```bash
  cd LiquidOnCpu
```

Build

```bash
  make
```

Run
```bash
  ./main
```
